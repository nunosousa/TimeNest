# Hardware

This directory contains the electronic design files for TimeNest.

## Contents

- schematics
- PCB layout
- component libraries
- manufacturing files

The main PCB includes:

- RP2040 microcontroller
- Ethernet MAC/PHY
- DAC for signal generation
- antenna driver
- LED drivers
- power supply circuitry

## Tools

The PCB is designed using KiCad.

## Status

Hardware design is currently in early development.
